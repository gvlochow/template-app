## Al usar este template para un proyecto nuevo
1. Repo nuevo desde "Use this template" en GitHub — nunca clonar template-app directo.
2. Cambiar "name" en package.json (no dejar el nombre del template — ya te pasó una vez).
3. Completar CLAUDE.md con el contexto real del proyecto.
4. Completar DESIGN.md (etapa 0.5) antes de escribir el primer componente.
5. .env.local con las claves reales de Supabase — nunca commitear.
6. Activar branch protection en main con el check de CI.
